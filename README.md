# Event Management System

![Java](https://img.shields.io/badge/Java-25-blue)
![Maven](https://img.shields.io/badge/Maven-3.x-green)

A desktop application for creating and managing events, venues, and attendee registrations. Originally developed as a school project using **NetBeans** and **MySQL**, and currently being modernized to use **IntelliJ IDEA**, **Maven**, and **SQLite**.

## Motivation

I am updating this project to learn modern build tools and get familiar with **SQLite** in preparation for a future project.

## Features

- Create and manage events
- Create and manage venues
- Register attendees to events

## Tech Stack

- **Java 25**
- **Maven**
- **SQLite**
